# 🪐 ArcMap - Identity Platform Portal

This is the frontend deployment for **ArcMap**, an on-chain identity protocol on the **Arc Network**. Users can connect their wallets to interact with the domain registration system.

## 🚀 Live Demo
The platform is hosted on Vercel. You can access it via the generated `.vercel.app` domain.

## 💻 Tech Stack
* **Frontend:** HTML5, Tailwind CSS, Ethers.js
* **Network:** Arc Network Testnet
* **Smart Contract:** 0xE78c2Da0a438e70542fd0d424471EC0eF1213E4A

## 🛠️ How to use
1. Connect your MetaMask wallet.
2. Ensure you are on the **Arc Testnet** network.
3. Search for your desired `.arc` domain.
4. Register using Testnet USDC.
